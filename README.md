## 注意

* あくまで作者の考える基準であり、一般的に正しいと主張するものではありません。  
* コーディングにまつわる作業全般を対象にしているため
  Java言語によるコーディング以外の観点のものも含んでいます。  
* 多様なバックグラウンドを持つエンジニアが意識を合わせてコーディングできるよう、
  Javaプログラマにとっては常識といえるものも入っていたり、
  あるいは一般的なJavaプログラミングにおいて普通なことと反する内容が
  書いてある可能性があります。
